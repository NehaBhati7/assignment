1. <html>: The root element of an HTML page, representing the entire document.
  2. <head>: Contains meta-information about the HTML document, such as the title, styles, and scripts.
  3. <title>: Defines the title that appears in the browser tab or window title bar.
  4. <body>: Contains the visible content of the HTML document, such as headings, paragraphs, images, and links.
  5. <h1> to <h6>: Define headings with different levels of importance, from most important (h1) to least important (h6).
  6. <p>: Defines a paragraph of text.
  7. <br>: Inserts a line break.
  8. <img>: Inserts an image into the document.
  9. <a>: Defines a hyperlink to another resource.
  10. <ul>: Defines an unordered list.
  11. <ol>: Defines an ordered list.
  12. <li>: Defines a list item within a list.
  13. <table>: Defines a table.
  14. <tr>: Defines a table row.
  15. <td>: Defines a table cell.
