BEM Naming Convention Example

This is the first paragraph with a highlighted style.
Block:
The block is the main component or container.
Example:
content represents the main section containing the paragraphs.

This is the second paragraph with a primary style.
Element:
The element is a part of the block.
Example:
content__paragraph represents the paragraphs within the content block.

This is the third paragraph with a secondary style.
Modifier:
A modifier defines a specific state or variation of an element.
Examples:
content__paragraph--highlighted applies a "highlighted" style.
content__paragraph--primary applies a "primary" style.
content__paragraph--secondary applies a "secondary" style.
